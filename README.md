# Описание

Архив моих модов, часть из которых я не выкладывал в мастерскую.
Любая херня, попадающая под категорию "не, ну это читы" тоже здесь.

## Brewmaster

Пиво случайных эффектов. Мод заброшен, я лишь тренировался в работе с Lua.
С некоторым шансом можно превраться в ползуна.

## Low TTK

Снижает здоровье у гуманоидов, выстрел в голову теперь летален.
Есть в [мастерской](https://steamcommunity.com/sharedfiles/filedetails/?id=2952982537&searchtext=Low+TTK), но морально устарел.
Проще установить альтенативу на Lua с меню настроек.

## Oxygen Reloader

Мод из QoL категории: быстро меняет кислородный баллон по нажатию кнопки.
Используется самописный алгоритм ~~говна~~ поиска подходящего баллона, вплоть до оценки качества.
Может работать с ошибками.

## Handcuffer Legacy

Залит сюда по просьбам трудящихся, я не выявил разницы в работе между [Handcuffer](https://steamcommunity.com/sharedfiles/filedetails/?id=3055574436) и Legacy версией.

## Blueshield Job

Добавляет профессию [офицера «Синий щит»](https://wiki.ss220.space/index.php/%D0%9E%D1%84%D0%B8%D1%86%D0%B5%D1%80_%C2%AB%D0%A1%D0%B8%D0%BD%D0%B8%D0%B9_%D1%89%D0%B8%D1%82%C2%BB).
Некое подобие шутки и действительно полезной профессии.
В мастерской _пока_ не появился.
