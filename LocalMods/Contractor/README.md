# Contractor

A Barotrauma mod that adds a contract system to the game. Players can accept and complete contracts for credits by eliminating creatures, poisoning crew members, or neutralizing hostiles.

## Features

- **Creature Contracts**: Accept contracts to eliminate specific creatures on the map. Progress is tracked automatically.
- **Poison Contracts**: Poison specified crew members with various toxins to earn rewards.
- **Elimination Contracts**: Hunt down hostile humans and eliminate them.
- **Dynamic Rewards**: Reward amounts scale with difficulty and number of targets.
- **Chat Commands**: Use in-game chat to interact with the contract system.

## Usage

1. Open the contract menu using the designated item or trigger.
2. Browse available contracts showing target type, reward amount, and difficulty.
3. Accept a contract by typing `/accept [number]` in chat.
4. Complete the objective to receive credits.

## Contract Types

| Type | Description |
|------|-------------|
| Creature | Kill specified creatures on the map |
| Poison | Apply poison to a specific crew member |
| Elimination | Neutralize hostile humans |

## Difficulty Colors

- Green: Easy (difficulty 1-2)
- Yellow: Medium (difficulty 3-4)
- Red: Hard (difficulty 5+)

## Requirements

- Barotrauma
- LocalMods folder installation

## Installation

1. Copy the `Contractor` folder to your `LocalMods` directory.
2. Enable the mod in the game's content package manager.
3. Start a new round or join a server to activate the mod.
